You need to configure your directory for this code to work, as there is another .txt file which comes along with the actual python code which contains all english words, and the code sifts through and collects all the five letter words in the file. Make sure that the directory is set to where you have the Python file and the .txt file in the same folder for the code to work properly. 

example:

You should have my Wordle.py and wordlist.txt in the same folder, and let's say that you have the folder containing both of those in the Documents section.

Your directory should look something like this:

Users/YOUR_NAME/Documents/Wordle

Enjoy!!!
